# 《明日方舟》抽卡模拟&数据分析

## 简介：
使用Python模拟《明日方舟》中各个卡池，并附上模拟的各卡池抽数统计数据。

目前支持卡池：
 - 标准/中坚寻访：StandardBanner类
 - 特殊寻访：SpecialBanner类
 - 限定寻访：LimitedBanner类

计划/施工中卡池：
 - 定向甄选
 - 联动寻访

本项目采用MIT协议，使用数据/代码请标注来源；勘误请联系作者或pr。
